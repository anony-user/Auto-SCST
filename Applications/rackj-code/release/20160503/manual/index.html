RACKJ

RACKJ (Read Analysis & Comparison Kit in Java) is a set of Java programs and Perl scripts that analyze and compare RNA-seq data made by NGS (Next-Generation Sequencing) technologies. In addition to RPKM (Reads Per Kbp per Million reads) values, RACKJ computes read counts for exons and splicing events. In so doing, it is feasible to compare two samples and identify genes with most significant difference in exon(splicing)-level.

In addition to processing RNA-seq data, RACKJ is also capable of processing DNA-seq data, including SNP/InDel calling, T-DNA insertion site detection, and genome rearrangement detection.

Download

  • The latest release of RACKJ is available at the download area of the RACKJ project.
  • RACKJ uses a very simple format to record gene locations and exon ranges. You may use the CanonicalGFF program (see here for example) or the SeqGeneMd program to generate a .cgff file and a .model for yourself. You may also download prebuild .cgff and .model files.
